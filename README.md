# Mortal-Kombat

# Make sure to turn your sound ON for better game experience.  

1. Select fighter 1
2. Select fighter 2
3. Press Attack button until you win or loose.
4. Once you win, select next opponent.
5. If you lose, you must restart.
6. HAVE FUN!
